[개념] display:inline-block의 특성display 속성은 요소가 화면에 어떻게 보이고, 다른 요소와 어떻게 상호작용하는지를 결정합니다. inline, block, inline-block의 차이점은 다음과 같습니다.특성display: inlinedisplay: blockdisplay: inline-block줄바꿈줄바꿈이 일어나지 않음 (글자처럼 취급)항상 새로운 줄에서 시작 (영역을 차지)줄바꿈이 일어나지 않음 (글자처럼 취급)크기 (width/height)적용 불가적용 가능적용 가능상하 여백 (margin-top/bottom)적용 불가적용 가능적용 가능좌우 여백 (margin-left/right)적용 가능적용 가능적용 가능주요 요소<span>, <a>, <img><div>, <p>, <h1>, <ul>, <li>(CSS로 지정해야 함)inline-block의 핵심 특성:inline 요소처럼 가로로 배치되면서, block 요소처럼 **너비(width), 높이(height), 그리고 상하 여백(margin/padding)**을 가질 수 있습니다.예제: inline, block, inline-block 비교HTML:<div class="container">
  <span class="inline-example">inline</span>
  <span class="inline-example">요소</span>
</div>

<div class="container">
  <div class="block-example">block</div>
  <div class="block-example">요소</div>
</div>

<div class="container">
  <div class="inline-block-example">inline-block</div>
  <div class="inline-block-example">요소</div>
</div>
CSS:/* 공통 스타일 */
.container {
  border: 1px solid #ccc;
  margin-bottom: 10px;
}
span, div {
  width: 150px;       /* 너비 시도 */
  height: 50px;      /* 높이 시도 */
  margin: 10px;        /* 여백 시도 */
  padding: 10px;
  background-color: #f0f0f0;
}

/* 1. inline (너비/높이/상하 여백이 무시됨) */
.inline-example {
  display: inline;
}

/* 2. block (너비/높이/여백 모두 적용, 줄바꿈 됨) */
.block-example {
  display: block;
}

/* 3. inline-block (너비/높이/여백 모두 적용, 줄바꿈 안 됨) */
.inline-block-example {
  display: inline-block;
}
2. [실습] 가로 내비게이션 메뉴 만들기display: inline-block을 활용하여 <li> 요소들을 가로로 정렬하고 크기를 지정할 수 있습니다.HTML:<nav class="main-nav">
  <ul>
    <li>메뉴 1</li>
    <li>
      메뉴 2
      <span class="new-badge">New!</span>
    </li>
    <li>메뉴 3</li>
    <li>메뉴 4</li>
  </ul>
</nav>
CSS:/* ul 태그의 기본 스타일(점, 들여쓰기) 제거 */
.main-nav ul {
  list-style-type: none; /* 리스트 앞의 점 제거 */
  padding-left: 0;
  margin: 0;
  background-color: #333; /* 메뉴바 배경색 (구분을 위해 추가) */
  text-align: center; /* 메뉴 항목들을 가운데 정렬 */
}

/* li 태그 (메뉴 항목) */
.main-nav li {
  display: inline-block;  /* 핵심: 가로 정렬 + 크기 적용 */
  
  /* 요청된 스타일 */
  width: 100px;
  height: 50px;
  background-color: #efefef;
  margin: 5px;

  /* 꾸미기 (수직 가운데 정렬) */
  line-height: 50px; /* height와 동일하게 맞춰 수직 정렬 */
  font-family: sans-serif;
  font-weight: bold;
  position: relative; /* New 배지 위치 조정을 위한 기준 */
}

/* New! 배지 */
.main-nav .new-badge {
  /* 요청된 스타일 */
  background-color: red;
  color: white;
  padding: 3px;

  /* 꾸미기 (배지처럼 보이도록) */
  font-size: 10px;
  line-height: 1; /* line-height 초기화 */
  border-radius: 3px;

  /* 배지 위치 (li 기준) */
  position: absolute;
  top: 5px;
  right: 5px;
}

[개념] background- 관련 속성배경을 제어하는 다양한 background- 속성들이 있습니다.속성설명주요 옵션background-color요소의 배경색을 지정합니다.transparent, #RRGGBB, rgb(), hsl(), blue 등background-image요소의 배경 이미지를 지정합니다.none, url('경로/이미지.jpg'), linear-gradient()background-repeat배경 이미지의 반복 방법을 지정합니다.repeat (기본값, 가로/세로 반복)  no-repeat (반복 없음)  repeat-x (가로로만 반복)  repeat-y (세로로만 반복)background-position배경 이미지의 시작 위치를 지정합니다.left top (기본값)  center center (정중앙)  right bottom (우측 하단)  50% 50% (% 값)  20px 10px (픽셀 값)background-size배경 이미지의 크기를 지정합니다.auto (기본값, 원본 크기)  cover (영역을 가득 채움, 이미지가 잘릴 수 있음)  contain (이미지가 잘리지 않게 영역 안에 다 들어옴)  100% 100% (너비/높이 % 지정)  200px 100px (픽셀 값 지정)background-attachment배경 이미지를 스크롤에 따라 움직일지 고정할지 지정합니다.scroll (기본값, 스크롤 시 같이 움직임)  fixed (뷰포트(화면)에 고정됨)  local (요소 내부 스크롤 시 같이 움직임)기본 예제 및 단축 속성(Shorthand)div {
  width: 300px;
  height: 200px;
  border: 1px solid black;

  /* 개별 속성 사용 */
  background-color: #f0f0f0;
  background-image: url('image.png');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

/* 단축 속성(shorthand) 사용 
  - 순서: color image repeat position / size
  - size는 position 뒤에 `/`로 구분하여 작성해야 합니다.
*/
div.shorthand {
  background: #f0f0f0 url('image.png') no-repeat center / cover;
}
4. [실습] '소개 카드' 만들기요청하신 조건에 맞는 card 클래스를 가진 div 예제입니다.HTML:<div class="card">
  <!-- 
    배경 이미지가 카드 역할을 하므로 
    내용이 비어있어도 크기(height)가 있어 표시됩니다.
    실제로는 내부에 텍스트 등이 들어갈 수 있습니다.
  -->
</div>
CSS:.card {
  /* 1. 크기 및 테두리 */
  width: 300px;
  height: 400px;
  border: 1px solid #ccc;

  /* 2. 배경색 (이미지 로딩 실패 시 대비) */
  background-color: #f0f0f0; 

  /* 3. 배경 이미지 */
  background-image: url('[https://picsum.photos/id/1/300/400](https://picsum.photos/id/1/300/400)');

  /* 4. 이미지 반복 안 함 */
  background-repeat: no-repeat;

  /* 5. 이미지 정중앙 배치 */
  background-position: center center;

  /* 6. 이미지가 영역을 가득 채우도록 (cover) */
  background-size: cover;
}

/* --- 참고: 단축 속성(shorthand) 사용 시 ---

  width: 300px;
  height: 400px;
  border: 1px solid #ccc;
  
  background: #f0f0f0 url('[https://picsum.photos/id/1/300/400](https://picsum.photos/id/1/300/400)') no-repeat center center / cover;
}
*/

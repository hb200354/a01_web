## 객관식 20문제 (4지선다형)

**[HTML/CSS: Box Model]**

**1. CSS 박스 모델에 대한 설명 중, '택배 상자' 비유가 올바르지 않은 것은?**
① Content (내용물) - 상자 안의 실제 물건 text, img, 
② Padding (뽁뽁이) - 물건 보호를 위한 안쪽 완충재 - 간격/방향
③ Border (상자) - 상자 자체의 두께(테두리) - 간격/방향/스타일/굵기
#④ Margin (거리) - 상자 안쪽의 개인 공간 - 간격/방향

**2. `padding` 속성에 대한 설명으로 올바르지 않은 것은?**
① 테두리(border)와 내용물(content) 사이의 안쪽 여백이다.
② '패딩 점퍼'처럼 안쪽을 빵빵하게 채우는 개념과 유사하다.
#③ `padding` 영역에는 요소의 배경색이 적용되지 않는다. 
	- padding 속성으로 색상을 지정하지 않지만 background-color 이 요소객체의 전체
	배경색상을 지정되어 표현할 수 있다.
④ `padding-left`, `padding-top` 등으로 방향을 지정할 수 있다.

**3. `margin` 속성에 대한 설명으로 가장 올바른 것은?**
#① 테두리(border) 바깥쪽의 여백으로, 다른 요소와의 거리를 의미한다.
② `margin` 영역에는 요소의 배경색이 함께 적용된다.
③ `margin`은 안쪽 여백을 의미하며 '뽁뽁이'에 비유된다.
④ `margin` 값은 `px`로만 설정해야 한다.

**4. `border-style` 속성의 값 중 '이중 실선'을 의미하는 것은?**
① solid
② dashed(각진) dotted(동글동글) 점선
③ groove 입체적인 효과
#④ double

**5. `border-style` 속성의 값 중 '파선(긴 점선)'을 의미하는 것은?**
① dotted
#② dashed
③ inset
④ solid

**6. `margin: 10px 20px;`와 같이 값을 2개만 설정했을 때, 올바른 해석은?**
#① 위/아래 10px, 좌/우 20px
② 위/아래 20px, 좌/우 10px
③ 위 10px, 좌/우 20px, 아래 10px
④ 위 10px, 오른쪽 20px, 나머지는 0

**7. `padding: 10px 20px 30px 40px;`와 같이 값을 4개 설정했을 때, 
      `40px`가 적용되는 방향은?**

**18. Oracle에서 `GRANT` 명령으로 사용자에게 부여하는 '핵심 열쇠 꾸러미(Role)' 중, 
	데이터베이스 접속을 위한 가장 기본적인 권한은 무엇인가?**
① RESOURCE
② DBA
#③ CONNECT
④ SYSTEM
1. 계정 생성
	CREATE USER 사용자이름 IDENTIFIED BY 비밀번호; 
	ex) CREATE USER USER01 IDENTIFIED BY 1234;
2. 접속권한 부여	
	
	GRANT 권한이름1, 권한이름2,... TO 계정
	기본 접속
	ex) GRANT CONNECT TO USER01;  (ROLE명)
	ex) GRANT CREATE SESSION TO USER01;
	-- 접속을 할 수 있으나, 테이블이나 자원들을 사용할 수 없음
	자원 사용
	EX) GRANT RESOURCE TO USER01;
	-- 테이블도 생성하고 자원들 사용할 수 있음..
	다른 사용자에 권한부여 권한(관리자권한)
	EX) GRANT DBA TO USER01;
3. 권한 제거
	REVOKE 권한이름1, 권한이름2,.... FROM 사용자이름;
	EX) REVOKE RESOURCE FROM USER01;	
4. 계정 삭제
	DROP USER 사용자이름; - 포함되어 있는 테이블이나 객체가 없을 때.
	DROP USER 사용자이름 CASCADE;	- 포함되어 있는 테이블이나 객체를 만들었을 때.
		같이 삭제 처리..
	EX) DROP USER USER01;

## 📝 객관식 문제 (총 20문항)

1.  제공된 코드의 주석에 따르면, `int a = 10;` 코드의 정확한 의미로 옳은 것은?
    (a) a는 10과 같다.
    (b) 10은 a와 같다.
    (c) 10이라는 값을 a라는 상자(변수)에 할당(저장)하라는 명령이다.
    (d) a라는 변수를 10으로 초기화할 수 있는지 확인한다.

2.  코드 예제 중 `String loc="수원";`에서 변수 `loc`의 데이터 타입(유형)은 무엇인가?
    (a) int
    (b) String
    (c) char
    (d) double

3.  증감 연산자 `level++`에 대한 설명으로 옳은 것은?
    (a) level의 값을 1 증가시킨다.
    (b) level의 값을 2 증가시킨다.
    (c) level의 값을 0으로 초기화한다.
    (d) level의 값을 1 감소시킨다.

4.  "변수 앞에 붙으면 전위(선불), 뒤에 붙으면 후위(후불)"라고 설명하는 연산자는?
    (a) =
    (b) +=
    (c) %
    (d) ++ 또는 --

5.  전위 증감 연산자 `++a` (선불)에 대한 설명으로 옳은 것은?
    (a) 먼저 1을 올리고, 그 다음에 1이 올라간 최신 a값을 사용한다.
    (b) a의 현재 값을 먼저 사용하고, 모든 일이 끝난 후 a의 값을 1 올린다.
    (c) a의 값을 2 올린다.
    (d) a의 값에 1을 더한 후 출력만 한다 (a의 값은 변하지 않는다).

6.  후위 증감 연산자 `a++` (후불)에 대한 설명으로 옳은 것은?
    (a) 먼저 1을 올리고, 그 다음에 1이 올라간 최신 a값을 사용한다.
    (b) a의 현재 값을 먼저 사용하고, 모든 일이 끝난 후 a의 값을 1 올린다.
    (c) a의 값을 2 올린다.
    (d) a의 값에 1을 더한 후 출력만 한다 (a의 값은 변하지 않는다).

7.  전위/후위 비교 예제에서 `int a = 5; int resultA = ++a;`가 실행된 후 `resultA`와 `a`의 값으로 옳은 것은?
    (a) resultA: 5, a: 5
    (b) resultA: 5, a: 6
    (c) resultA: 6, a: 6
    (d) resultA: 6, a: 5

8.  전위/후위 비교 예제에서 `int b = 5; int resultB = b++;`가 실행된 후 `resultB`와 `b`의 값으로 옳은 것은?
    (a) resultB: 5, b: 5
    (b) resultB: 5, b: 6
    (c) resultB: 6, b: 6
    (d) resultB: 6, b: 5

9.  `count` 예제 중 `System.out.println("현재 카운트3:"+(count++));`가 실행될 때, 콘솔에 출력되는 값은? 
    (이전 라인에서 `count`는 7이 되었음)
    (a) 6
    (b) 7
    (c) 8
    (d) 9

10. 편의점 재고 예제에서 `int reportA = apples++;` (apples=5)가 실행된 후 `reportA`의 값은?
    (a) 4
    (b) 5
    (c) 6
    (d) 10

11. 편의점 재고 예제에서 `int reportB = ++bananas;` (bananas=5)가 실행된 후 `reportB`의 값은?
    (a) 4
    (b) 5
    (c) 6
    (d) 10

12. 편의점 재고 예제에서 `int totReport = reportA+reportB;`가 실행된 후 `totReport`의 최종 값은?
    (단, `reportA`는 5, `reportB`는 6이 할당됨)
    (a) 10
    (b) 11
    (c) 12
    (d) 13

13. "게으름뱅이 연산자"라고 소개된 복합 대입 연산자로 옳지 않은 것은?
    (a) +=
    (b) -=
    (c) ++
    (d) *=

14. `score += 10;` 코드와 완전히 동일한 의미를 가진 코드는?
    (a) `score = score + 10;`
    (b) `score = 10;`
    (c) `score++;`
    (d) `score = score * 10;`

15. 용돈 예제에서 `int myMoney = 10000; int snack = 1500; myMoney -= snack;`가 실행된 후 
`myMoney`의 값은?
    (a) 10000
    (b) 11500
    (c) 8500
    (d) 1500

16. 예제 `System.out.println("총 밀가루:"+ flourA + flourB);`가 "총 밀가루:10050"으로 출력되는 이유로 옳은 것은? (단, `flourA`=100, `flourB`=50)
    (a) 100과 50이 곱해졌다.
    (b) 100과 50이 더해졌다.
    (c) 문자열 "총 밀가루:" 뒤에 숫자 100과 50이 순서대로 붙어 문자열이 되었다.
    (d) 괄호()가 없어서 오류가 발생한다.

17. `int a = 10; int b = 3; int result = a / b;` 코드가 실행된 후 `result`의 값은? 
("정수 셰프의 법칙" 참고)
    (a) 1
    (b) 3
    (c) 3.333
    (d) 4

18. `int a = 10; int b = 3; int result = a % b;` 코드가 실행된 후 `result`의 값은? ("모듈로" 참고)
    (a) 1
    (b) 3
    (c) 0
    (d) 3.333

19. 슈퍼마리오 시간 변환 예제에서 `int totalSeconds = 500;`일 때, `minutes` (분)를 구하기 위한 올바른 연산은?
    (a) `totalSeconds % 60`
    (b) `totalSeconds / 60`
    (c) `totalSeconds * 60`
    (d) `totalSeconds - 60`

20. 달걀 포장 예제에서 `int totalEggs = 115; int eggsPersCarton = 30;`일 때, 남은 달걀(`remainEggs`)을 구하기 위한 올바른 연산은?
    (a) `totalEggs / eggsPersCarton`
    (b) `totalEggs * eggsPersCarton`
    (c) `totalEggs - eggsPersCarton`
    (d) `totalEggs % eggsPersCarton`

## 🔑 주관식 문제 (총 5문항)

1.  제공된 코드 주석에 따르면 `a = 10`은 "a와 10은 같다"라는 의미가 아니라고 설명합니다. `a = 10`의 정확한 의미는 무엇인지 '할당'이라는 용어를 사용하여 기술하시오.

2.  전위 증감 연산자(`++a`)와 후위 증감 연산자(`a++`)의 차이점을 "선불", "후불" 용어를 사용하여 설명하시오.

3.  `hp -= 20;`과 같이 연산과 대입을 합쳐놓은 연산자를 "게으름뱅이 연산자"라고 부른 이유가 무엇인지 
`hp = hp - 20;` 코드와 비교하여 설명하시오.

4.  "정수 셰프의 법칙"이라고 설명한 `int / int` (정수 나누기 정수) 연산의 특징을 `10 / 3`의 결과를 
    예시로 들어 설명하시오.

5.  `%` (모듈로) 연산자는 무엇을 구하는 연산자인지, 그리고 이 연산자가 슈퍼마리오 시간 예제에서 어떻게 사용되었는지 
    기술하시오.
